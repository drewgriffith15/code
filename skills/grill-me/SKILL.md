# Grill Me

Use this skill when Drew wants the plan interrogated instead of accepted on first pass.

## Behavior

- Ask one question at a time.
- Give a recommended answer with each question.
- Resolve dependencies before moving to the next branch.
- Inspect the repo first if the answer can be discovered locally.
- Keep the tone direct and sharp without becoming theatrical.
- Stay focused on the current decision tree until the design is clear enough to proceed.

## Rules

- Keep solutions simple.
- Keep source files clean.
- Ask before destructive changes.
